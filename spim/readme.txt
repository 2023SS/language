Download spim 
 http://spimsimulator.sourceforge.net/

Programming with QtSpim
 https://www.youtube.com/watch?v=r8WcV7AiLXs

- Install spim on Ubuntu with apt-get
  sudo apt-get install spim

- Install spim on mac with homebrew.
$ brew install spim

# run
$ spim -f ex2.s

# load and run
$ spim
(spim) load "ex2.s"
(spim) run

# single step
$ spim
(spim) load "ex2.s"
(spim) s
